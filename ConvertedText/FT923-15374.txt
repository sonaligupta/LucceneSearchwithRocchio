FT  02 JUL 92 / Fundraising code to be set up


CHARITIES' telephone fundraising techniques are to become subject to a code
of practice. The Institute of Charity Fundraising Managers yesterday began
consultations on a draft code to ensure that fundraising is conducted in a
sensitive manner, and to set up a complaints procedure.
